Our Recommendations

In this page, we present our recommendations for COMP1950 course website.

Layout Design

comp1950 website does not have an attractive layout design which is important to generate enthusiasm for students.

For instance, the website doesn't use advanced CSS properties such as box shadows, gradients, transitions, etc, even though it teaches these concepts.

Our proposed site uses attractive fonts, harmonious colors, and CSS techniques such as box shadows. Also, we use accordions for each header with sticky positions so that users can hide or show the content without being overwhelmed by the total textual content on the page.

Navigation is always visible and takes up precious horizontal space which is needed for textual content of the site.

Our proposed site hides the navigation while the user is always able to click the menu button to see the it; therefore, the horizontal space is used more efficiently.

SEO

The website does not use search engine optimization techniques.

Our proposed site uses < meta name="description" > to effectively summarize the content of the page.

A compound title for the pages is created which includes the word BCIT as company name.

A custom error 404 page is created which helps to improve SEO rankings.

For future implementations: GZip compression and cache control can be used to increase loading time of pages.

Content Organization

Exercises and homework are distributed between the notes which is helpful during lectures. However, it would be benefitial to students to have access to all of them in one location for reviewing.

In our proposed site map, a separate link to all exercises, homework, and useful resource links (such as CSS validator, border box generator, etc) is provided.

For future implementations: an online quiz will help to achieve a paperless comp1950 course.

For future implementations: A search feature for the entire content of the site can be useful. In our proposed layout, we have provided a placeholder for this search input.

Other

For optimum effectiveness, our proposed website has three different layouts for

  • screen
  • print
  • projection

For future implementations: A style switching plugin (such as stylesheet switcher) can be used to change the style with one click.

For future implementations: An online accompanying forum (with login credentials) for COMP1950 can be really helpful where students can share useful links, information and ask questions.
